🇰🇷 사용 방법 (Korean)
프로그램 개요

이 프로그램은 지정한 **감시 폴더(Watch Folder)**에 이미지가 추가되면
자동으로 **직선 보정(Straighten)**을 적용하여
**출력 폴더(Output Folder)**에 저장하는 도구입니다.

Apply 버튼을 누르면 계속 적용되며,
Disable 버튼을 누르면 즉시 중지됩니다.
---
사용 방법
1. 프로그램 실행

StraightenTool.exe 실행

상단 상태 표시가 OFF 상태로 시작합니다.
---
2. 감시 폴더 설정 (Watch Folder)

Select 버튼을 눌러

원본 이미지가 저장되는 폴더를 선택합니다.

📌 이 폴더에 이미지가 들어오면 자동 처리됩니다.
---
3. 출력 폴더 설정 (Output Folder)

Select 버튼을 눌러

보정된 이미지가 저장될 폴더를 선택합니다.
---
4. 모드 선택

auto
→ 이미지의 기울기를 자동으로 분석하여 보정합니다.

manual
→ 사용자가 입력한 각도로 고정 회전합니다.
---
5. 수동 각도 입력 (Manual Angle)

manual 모드에서만 사용

예시:

5 → 시계 방향 5도

-3 → 반시계 방향 3도
---
6. 적용 (Apply)

Apply 버튼 클릭

상태가 ON (초록색)으로 변경됩니다.

이후 감시 폴더에 들어오는 이미지는 자동으로 처리됩니다.
