Overview

This program automatically applies image straightening to images added to a selected Watch Folder
and saves the processed results to an Output Folder.

When you press Apply, the process runs continuously.
Press Disable to stop it immediately

---
Instructions
1. Launch the Program

Run StraightenTool.exe

The status starts as OFF.

---

Click Select next to Watch Folder
